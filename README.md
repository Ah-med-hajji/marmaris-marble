# Marmaris Marble — Landing Page

A professional, minimalist landing page for a Tunisian marble company based in Thala, Kasserine, Tunisia.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **i18n:** next-intl (French / Arabic with RTL)
- **Icons:** react-icons
- **Fonts:** Playfair Display, Inter, Noto Sans Arabic (Google Fonts)
- **SEO:** next-sitemap, JSON-LD structured data, Open Graph, Twitter Cards
- **Deployment:** Vercel

## Features

- Fully responsive (mobile-first)
- French (default) and Arabic language support with automatic RTL
- Smooth scroll navigation
- Scroll-triggered animations
- Product gallery with hover effects
- WhatsApp & Facebook contact buttons
- Embedded Google Maps
- Full SEO: meta tags, sitemap.xml, robots.txt, JSON-LD, hreflang, canonical URLs

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production (generates sitemap automatically)
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Customization

Search for `TODO` comments in the codebase to find placeholders that need to be replaced:

- WhatsApp phone number (`https://wa.me/21600000000`)
- Facebook page URL (`https://facebook.com/marmarismarble`)
- Telephone number in JSON-LD (`+216-00-000-000`)
- `SITE_URL` in `src/app/[locale]/layout.tsx` — set to your production domain

## SEO Configuration

### Sitemap & Robots.txt
- `robots.txt` is in `public/` and allows all crawlers
- `sitemap.xml` is auto-generated after each build via `next-sitemap`
- Update `next-sitemap.config.js` with your production URL

### Google Search Console Verification
After deploying to production:
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property (e.g., `https://marmarismarble.com`)
3. Verify ownership using one of:
   - **HTML tag:** Add the meta tag to `src/app/[locale]/layout.tsx` in the `metadata` object under `verification: { google: 'YOUR_CODE' }`
   - **DNS record:** Add a TXT record to your domain
4. Submit your sitemap URL: `https://marmarismarble.com/sitemap.xml`

### Structured Data
JSON-LD `LocalBusiness` schema is embedded in the layout. It includes:
- Business name, address (Thala, Kasserine), geo coordinates
- Business hours, social links, image
- Update the placeholder values (telephone, Facebook) before going live

## Deployment

This project is configured for [Vercel](https://vercel.com) deployment:

1. Push to GitHub
2. Import the repository on Vercel
3. Deploy — no additional configuration needed
4. Set `SITE_URL` environment variable in Vercel to your production domain
